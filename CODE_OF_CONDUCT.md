# Code of Conduct

The Liquid project is committed to providing a welcoming, harassment-free
environment for everyone — regardless of experience level, background, or
identity.

## We adopt the Contributor Covenant

Liquid follows the **Contributor Covenant, version 2.1**, the de-facto
industry-standard code of conduct used by Rust, Flutter, Node.js,
Kubernetes, and most major open-source projects.

The full text is published at:

> <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

We adopt that text by reference rather than copying it inline, so that
updates to the standard reach this project without manual sync. The
canonical version always wins.

## Scope

This Code of Conduct applies in every project space — the GitHub
repository, issues, pull requests, discussions, code review comments,
project chat channels (when they open), and any in-person or virtual
event where someone is acting as a representative of the project.

## Enforcement contact

Reports of unacceptable behaviour may be sent to the project maintainers
at:

> **conduct@liquid-project.org** *(placeholder — to be replaced with the
> active maintainer alias before the project announces general
> availability; until then, send a direct message via GitHub to one of
> the repository administrators).*

All reports will be reviewed and investigated promptly and fairly. The
maintainers are obligated to respect the privacy and security of anyone
reporting an incident.

## Enforcement actions

Maintainers follow the **Contributor Covenant Enforcement Ladder** when
deciding consequences for any action they deem in violation of this Code
of Conduct:

> <https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines>

The ladder ranges from a private written warning, through temporary bans
from interaction, up to a permanent ban from the community. The specific
response is calibrated to the severity and pattern of the violation.
